What's new in BMS v20?

  • New germplasm list manager: A brand-new germplasm list manager is available. Easily search for germplasm lists, customize the columns, and describe the entries using “entry details” variables.
    Learn more
  • Upload photos and files: You can now upload files and photos and associate them to your germplasm and/or plots. If you’re hosting your BMS instance on-premises, reach your IBP point of contact to configure this setting and evaluate hosting options!.
    Learn more
  • Merge germplasm: Duplicated germplasm can now be merged into a target one, the BMS will automatically delete the non-wanted germplasm records and transfer inventory, attributes, files and other information to the selected target record. This action is now available in the Manage germplasm module.
    Learn more
  • Misc:
    • We'd like your feedback! We will be routinely publishing the BMS features under consideration. You can vote, provide insights and submit new ideas in our ProductBoard page.
    • Additionally, you can check what's in the roadmap for the next iterations here.

Notes de version v20

  • Nouveau gestionnaire de liste de germoplasmes:Un tout nouveau gestionnaire de listes de germoplasmes est disponible. Recherchez facilement des listes de germoplasmes, personnalisez les colonnes et décrivez les entrées à l'aide des variables "entry details".
    Learn more
  • Télécharger des photos et des fichiers:Vous pouvez maintenant télécharger des fichiers et des photos et les associer à vos germoplasmes et/ou plots. Si vous hébergez votre instance BMS sur site, contactez votre point de contact IBP pour configurer ce paramètre et évaluer les options d'hébergement!.
    Learn more
  • Merge germplasm: Les germoplasmes dupliqués peuvent désormais être fusionnés dans un enregistrement cible. Le BMS supprimera automatiquement les enregistrements de germoplasmes non désirés et transférera l'inventaire, les attributs, les fichiers et autres informations vers l'enregistrement cible sélectionné. Cette action est maintenant disponible dans le module Manage Germplasm.
    Learn more
  • Misc:
    • Nous aimerions votre feedback! Desormais, nous publierons régulièrement les fonctionnalités BMS envisagées. Vous pouvez voter, fournir des informations et soumettre de nouvelles idées sur notre page ProductBoard (vous pouvez le faire en francais!).
    • De plus, vous pouvez vérifier le contenu de la feuille de route pour les prochaines itérations ici.
