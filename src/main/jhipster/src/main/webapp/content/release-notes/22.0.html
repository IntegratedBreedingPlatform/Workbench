What’s new in BMS v22?

Revamped “Germplasm & checks” tab in studies

A much more flexible germplasm and checks tab is coming to the study manager. You can now bring germplasm attributes, names and pedigree information as columns. You can also easily annotate your entries in a custom fashion by defining and using Entry Details variables. All of this information is entirely customizable and readily available for a more comprehensive field book.

Learn more 

SSA results where you need them

New tabs are coming to the study manager so that you are able to see the estimated means results coming from of the single site analysis (SSA) along with it’s summary statistics, all without leaving your study in the study manager.

Learn more

Annotate inventory lots using the ontology

Want to describe your inventory lots? Add tags? Capture measurements? You can now use the ontology manager and the variable type Inventory Attribute to capture any information that you’d like about your lots. File upload/download is also supported!

Learn more 

Assortment of small improvements

  • You can now delete field maps and geo references from a study

  • Performance improvements and bug fixes.

Quoi de neuf dans BMS v22 ?

Onglet « Germoplasme et contrôles » remanié dans les études

Un onglet “Germplasm & Checks” beaucoup plus souple arrive les études. Vous pouvez maintenant apporter des attributs de germoplasme, des noms et des informations généalogiques sous forme de colonnes. Vous pouvez également facilement annoter vos entrées de manière personnalisée en définissant et en utilisant des variables “Entry Details”. Toutes ces informations sont entièrement personnalisables et facilement disponibles pour un fieldbook plus complet.

Learn more 

Des résultats SSA là où vous en avez besoin

De nouveaux onglets arrivent dans le gestionnaire d'études afin que vous puissiez voir les résultats moyens estimés provenant de l'analyse de site unique (SSA) ainsi que ses statistiques récapitulatives, le tout sans quitter votre étude dans le gestionnaire d'études.

Learn more

Annoter les lots d'inventaire à l'aide de l'ontologie

Vous souhaitez décrire vos lots d'inventaire ? Capturer des observations ? Vous pouvez maintenant utiliser le gestionnaire d'ontologie (Manage Ontologies) et le type de variable “Inventory Attribute” pour capturer toutes les informations que vous souhaitez sur vos lots. Le téléchargement/téléchargement de fichiers est également disponible !

Learn more 

Assortiment de petites améliorations

  • Vous pouvez maintenant supprimer des fieldmaps et des références géographiques d'une étude

  • Améliorations des performances et corrections de bogues.
